Goals

The various DIVA input (data, contours) and output files (mesh, analysis) can be easily displayed on an interactive map using Leaflet. Each layer can be made activated separately. The following example illustrates this with data from the Black Sea.

Processing

The file are simply transformed to geoJSON format using Python modules. For the analysis field, the iso-contours are turned into a multi-polygon to which a style is added.

In the toolbox DivaPythonTools, we provide the functions to convert each type of DIVA input or output into a geoJSON.
